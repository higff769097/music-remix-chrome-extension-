# music-remix-chrome-extension-
chrome extension works on the youtube music for saving the preferences of the user , helps the user to listen favourite segement of the song.

download all files include images.
Opening the Chrome browser on your PC.
Navigating to the Chrome Extensions page by entering `chrome://extensions` in the address bar.
Enabling Developer Mode by toggling the switch at the top right corner of the Extensions page.

click load unpacked upload folder downloaded.

click on the extension symbol on the top right of the chrome browser click on popup and add the segement you want to listen the next time you play the song.
